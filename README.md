
  # Minimalist Photo Archive Interface

  This is a code bundle for Minimalist Photo Archive Interface. The original project is available at https://www.figma.com/design/GR17pQmZWyabsQnTkcfECe/Minimalist-Photo-Archive-Interface.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  Run `python face_counter.py` to start the face counter.
